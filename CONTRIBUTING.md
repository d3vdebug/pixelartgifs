## How to Suggest a GIF

### Step-by-Step Process

1. **Find or Create a GIF**: Locate a pixel art GIF that fits our guidelines, or create your own
2. **Check Guidelines**: Ensure the GIF meets our [submission guidelines](#gif-guidelines)
3. **Open an Issue**: Create a new issue on GitHub with the subject "Add [GIF Name] to collection"
4. **Attach the GIF**: Upload the GIF file directly to the issue (GitHub supports GIF uploads)
5. **Provide Details**:
   - Brief description of the GIF
   - Source/attribution if not original
   - Suggested category or theme
   - Any special notes

### Example Issue Template

```
Title: Add Retro Coding Animation to Collection

Description:
I'd like to suggest this pixel art GIF of a character coding on a retro computer.

GIF: [attach file]

Source: Created by [artist name] on [platform] - [link if available]
Category: Characters/Scenes
Notes: This would fit well with developer-themed profiles
```

## GIF Guidelines

### Technical Requirements

- **Format**: GIF only (animated preferred)
- **File Size**: Under 5MB
- **Resolution**: 200x200px or smaller for profile compatibility
- **Style**: Pixel art aesthetic (8-bit, 16-bit style)
- **Content**: Appropriate for all audiences
- **Originality**: Original creations preferred, or properly attributed

### Content Guidelines

- Fun, creative, and engaging animations
- Suitable for GitHub profile READMEs
- Retro/pixel art style
- Professional and appropriate content

## What Happens Next

1. **Review**: The maintainers will review your suggestion
2. **Approval**: If approved, the GIF will be added to the collection
3. **Update**: The README will be updated with the new GIF and its usage code
4. **Notification**: You'll be notified when your suggestion is added

## Important Notes

- **Attribution**: If the GIF is not original, please provide proper attribution
- **Ownership**: The repository maintainer does not claim ownership of the GIFs - this is just a curated list for accessibility
- **Quality**: We aim to maintain high quality and relevance in the collection

## Reporting Issues

If you find any issues with the repository:

- **Broken Links**: Report if any GIF links are not working
- **Incorrect Information**: Let us know about any errors in descriptions or attributions
- **Suggestions**: Ideas for improving the repository structure or documentation

## Questions?

If you have questions about contributing:

- Check existing [Issues](https://github.com/d3vdebug/pixelartgifs/issues)
- Start a [Discussion](https://github.com/d3vdebug/pixelartgifs/discussions)
- Contact the maintainers

## Recognition

Contributors who suggest accepted GIFs will be:

- Mentioned in the commit messages when GIFs are added
- Recognized in the repository's acknowledgments
